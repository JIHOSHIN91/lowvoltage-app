# 구글 플레이 스토어 등록 초안

## 앱 이름 (30자 이내)
전력설비 점검공사 현장관리

## 간단한 설명 (80자 이내)
전력설비 점검공사 현장 데이터 입력·점검표 작성·사진대장 관리를 한 번에

## 상세 설명 (4000자 이내)
전력설비 점검공사 현장관리는 지상변압기, 저압접속함, 입상점, 분전함, 지상개폐기 등
전력설비 점검공사 현장에서 필요한 자료 입력, 점검표 작성, 사진 촬영·관리를 한 번에
처리할 수 있는 현장관리 도구입니다.

주요 기능
- 엑셀 설비 목록 업로드 → 설비별 점검 리스트 자동 생성
- 설비 구분별 맞춤 점검표(기본사항 자동입력 + 적합/부적합 판정)
- 촬영한 사진 자동 파일명 지정 및 사진대장 관리
- 점검표를 원본 양식 그대로 PDF로 출력
