# 개인정보처리방침 (전력설비 점검공사 현장관리 앱)

최종 수정일: 2026-07-20

본 앱("전력설비 점검공사 현장관리")은 전력설비 점검공사 현장 데이터 입력, 점검표 작성, 사진 관리를 지원하는 도구입니다.

## 1. 수집하는 정보
본 앱은 별도의 서버를 운영하지 않으며, 사용자가 앱에 업로드하거나 입력하는 모든 데이터(엑셀 백데이터, 점검표 입력값, 촬영한 사진)는 **사용자의 기기(브라우저) 내에서만 처리**되고, 외부 서버로 전송되거나 저장되지 않습니다.

## 2. 카메라 권한
사진대장 기능에서 사용자가 직접 촬영을 선택했을 때에만 기기의 카메라에 접근합니다. 촬영된 사진은 기기 내에서만 처리되며, 사용자가 직접 다운로드하기 전까지 외부로 전송되지 않습니다.

## 3. 정보의 제3자 제공
본 앱은 수집한 정보가 없으므로 제3자에게 제공하지 않습니다.

## 4. 데이터 보관 및 삭제
앱을 새로고침하거나 종료하면 입력된 데이터는 기기 메모리에서 사라집니다(별도 저장 기능을 사용하지 않는 한). 사용자는 언제든 브라우저 데이터를 삭제하여 저장된 내용을 제거할 수 있습니다.
